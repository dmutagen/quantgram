# 🪐 Quantgram

[![License](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)
[![Electron](https://img.shields.io/badge/Electron-v34.0.0-informational.svg)](https://electronjs.org)

**Quantgram** — это усовершенствованный, кастомный десктопный клиент для Telegram, построенный на базе Electron. Проект ориентирован на максимальную конфиденциальность, продвинутую стилизацию и производительность, предлагая гибкие инструменты для настройки рабочего окружения мессенджера под себя.

---

## ✨ Ключевые возможности

### 🔒 Безопасность и Приватность
* **Блокировщик рекламы**: Автоматическое скрытие спонсируемых сообщений и рекламных объявлений в каналах на уровне CSS.
* **Focus Protection (Blur-on-Blur)**: Размытие окон чатов, списков пользователей и сообщений при потере фокуса приложением для защиты от посторонних глаз.

### 🔔 Умные уведомления
* **Popup Toasts**: Всплывающие уведомления в углах экрана с гибкой настройкой длительности отображения (от 1 до 15 секунд).
* **Списки исключений и включений**: Возможность фильтровать уведомления по белым и черным спискам ключевых слов/имен пользователей.
* **Кастомные звуки**: Cyber Chime звуковые эффекты, синтезируемые через Web Audio API, или воспроизведение локальных `.wav` файлов с регулировкой громкости.
* **История уведомлений**: Удобный лог последних 50 событий прямо внутри панели управления.

### 🎨 Дизайн и Оформление (Material 3)
* **Панель управления**: Интуитивно понятный двухколоночный интерфейс настроек с боковым меню быстрого перехода.
* **Плавный скролл и Scrollspy**: Интегрированное отслеживание активного раздела с помощью высокопроизводительного `IntersectionObserver` (без просадок FPS).
* **Глобальный поиск**: Умный поиск по всем параметрам, описаниям и тегам настроек в реальном времени.
* **Компактный режим (Sidebar)**: Быстрое переключение в узкое окно, закрепляемое поверх других окон (Always on Top) для удобного фонового общения.
* **Кастомный CSS**: Встроенный live-редактор стилей для полной персонализации Telegram Web.
* **Настройка шрифтов**: Поддержка популярных семейств шрифтов (Inter, Outfit, Roboto, Google Sans, JetBrains Mono) или выбор любого системного шрифта.

### 🛠 Встроенные утилиты
* **Инлайн-переводчик**: Мгновенный перевод входящих сообщений (русский/английский) «на лету» с помощью бесплатного API Google Translate.
* **Быстрые ответы (Шаблоны)**: Удобная горизонтальная панель со списком заготовок фраз над полем ввода для быстрой отправки в один клик. Поддерживает плейсхолдер `{name}`.
* **Встроенный блокнот (Scratchpad)**: Выдвижная боковая панель для записей и заметок с автосохранением в локальное хранилище.
* **Интеграция с локальными LLM**: Поддержка умных суммаризаций диалогов и генерации ответов через Ollama, OpenAI, Gemini или Claude.

## 🚀 Системная интеграция и протокол `tg://` (Deep Linking)

Quantgram регистрирует себя в качестве глобального системного обработчика ссылок `tg://` на всех ОС:
* **Windows & macOS**: Нативная интеграция на уровне реестра и `Info.plist`. При клике по ссылке вида `tg://resolve?domain=username` на веб-сайтах, операционная система автоматически передает вызов Quantgram.
* **Linux**: Генерация и обновление валидного файла `.desktop` (с поддержкой `MimeType=x-scheme-handler/tg;` и аргумента `%u`), обеспечивающего интеграцию в лаунчеры сред GNOME, KDE, XFCE и др.
* **Single Instance Lock**: Предотвращает запуск дублирующих процессов. Если приложение уже запущено, клик по ссылке `tg://` сфокусирует основное окно и выполнит переход внутри него без перезагрузки интерфейса.

## 🛠 Техническая архитектура

* **[main.js](main.js)**: Главный процесс (Main Process). Отвечает за жизненный цикл Electron, блокировку повторных запусков, управление нативным окном, системные треи, иконки и глубокую системную регистрацию протоколов.
* **[index.html](index.html)**: Каркас интерфейса. Содержит кастомный кастомный фрейм окна (Title Bar), боковую панель диагностики и контейнер `<webview>` для изоляции Telegram Web. Принимает события глубоких ссылок и перенаправляет их в веб-клиент.
* **[preload.js](preload.js)**: Скрипт предварительной загрузки (Preload). Инжектируется непосредственно в контекст Telegram Web. Отвечает за рендеринг кастомной панели управления, автозамену текста «Telegram» -> «Quantgram», перехват кликов для внутреннего роутинга, интеграцию ИИ и CSS стилей.
* **[style.css](style.css)**: Стили оформления основного контейнера Electron и нативного фрейма.
* **[inject.css](inject.css)**: Стили-переопределения, инжектируемые внутрь Telegram Web для изменения цветовой палитры под фирменный стиль Quantgram.

## 📦 Быстрый старт

### Требования
* **Node.js** (версии 16 и выше)
* **npm** (поставляется вместе с Node.js)

### Установка и запуск

1. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/dmutagen/quantgram.git
   cd quantgram
   ```

2. Установите зависимости:
   ```bash
   npm install
   ```

3. Запустите приложение:
   ```bash
   npm start
   ```
